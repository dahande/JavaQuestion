Javaの基礎

問題1: インスタンス生成

次のうち、Javaでインスタンスを生成するための適切なキーワードはどれですか？

  • A) new
  • B) class
  • C) static
  • D) void

正解はA) newです。Javaでは、クラスからオブジェクト（インスタンス）を生成する際に「new」キーワードを使用します。

問題2: インスタンス変数とローカル変数

Javaでクラスのインスタンス変数とローカル変数の違いは何ですか？

  • A) インスタンス変数はstatic修飾子を持つが、ローカル変数は持たない。
  • B) インスタンス変数はクラス内のどこからでもアクセス可能だが、ローカル変数は定義されたブロック内でのみ有効。
  • C) インスタンス変数はfinal修飾子を持つが、ローカル変数は持たない。
  • D) インスタンス変数はクラスのメソッドでのみ使用可能だが、ローカル変数は全てのメソッドで使用可能。

正解はB) インスタンス変数はクラス内のどこからでもアクセス可能ですが、ローカル変数は定義されたブロック内でのみ有効です。

問題3: インスタンスの初期化

Javaでクラスのインスタンスを初期化するために必要なものは何ですか？

  • A) クラス名
  • B) メソッド名
  • C) newキーワード
  • D) インターフェース

正解はC) newキーワードです。Javaではインスタンスを生成する際には「new」キーワードを使用します。

問題4: インスタンス生成の流れ

次のJavaコードのうち、正しいインスタンス生成の流れを示しているものはどれですか？

class Car {
    String brand;

    Car(String b) {
        brand = b;
    }
}

public class Main {
    public static void main(String[] args) {
        Car myCar = new Car("Toyota");
        myCar.start();
    }
}
  • A) Carクラスのインスタンスが生成され、その後myCarのメソッドが呼び出される。
  • B) mainメソッドが呼び出され、その後Carクラスのインスタンスが生成される。
  • C) インスタンスが生成される前にmainメソッドが実行される。
  • D) myCarのインスタンス生成の後、Carクラスのメソッドが呼び出される。

正解はB) mainメソッドが呼び出され、その後Carクラスのインスタンスが生成されるです。Javaではmainメソッドがプログラムの実行を開始し、インスタンスはその後に生成されます。
