Javaの基礎

問題1: メソッド

Javaでメソッド（method）とは何ですか？

  • A) クラスのインスタンスを生成するための特別な関数。
  • B) クラスで定義された処理の塊で、特定の動作を実行するために使用される。
  • C) クラスの静的なプロパティを操作する関数。
  • D) クラスのコンストラクタを呼び出す特別な処理。

正解はB) クラスで定義された処理の塊で、特定の動作を実行するために使用されることです。メソッドはクラス内で定義され、オブジェクト指向プログラミングにおいてオブジェクトの振る舞いを定義します。

問題2: メソッドの定義

Javaでメソッドを定義する際、必須ではない要素はどれですか？

  • A) 戻り値の型。
  • B) メソッド名。
  • C) 引数リスト。
  • D) メソッドのアクセス修飾子。

正解はD) メソッドのアクセス修飾子です。メソッドの定義では、戻り値の型、メソッド名、および必要に応じて引数リストを指定しますが、アクセス修飾子は省略可能です。

問題3: メソッドのオーバーロード

Javaでメソッドをオーバーロードするとは何ですか？

  • A) メソッド内で別のメソッドを定義すること。
  • B) 1つのクラスに同じ名前のメソッドを複数定義すること。
  • C) メソッドの引数の型を変更すること。
  • D) メソッドの実装を一部上書きすること。

正解はB) 1つのクラスに同じ名前のメソッドを複数定義することです。メソッドのオーバーロードにより、同じ名前のメソッドを異なる引数リストで定義できます。

問題4: メソッドの戻り値

Javaのメソッドで戻り値を持たない場合、その戻り値の型は何ですか？

  • A) void
  • B) null
  • C) int
  • D) String

正解はA) voidです。Javaでメソッドが戻り値を持たない場合、そのメソッドの戻り値の型としてvoidを指定します。

問題5: メソッドの特性

Javaのメソッドの特性として正しいものはどれですか？

  • A) 1つのメソッド内で複数の戻り値を返すことができる。
  • B) メソッドは必ずしもクラス内に定義する必要がある。
  • C) メソッドは静的に宣言された変数のみを操作できる。
  • D) メソッドはクラス内のデータを操作するために利用される。

正解はD) メソッドはクラス内のデータを操作するために利用されることです。メソッドはクラス内で定義され、そのクラスの属性や状態を操作するために使用されます。
