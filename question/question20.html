問題91: オーバーライドのルール

次のコードの実行結果はどうなりますか？

class Parent {
    void show() {
        System.out.println("Parent");
    }
}

class Child extends Parent {
    @Override
    void show() {
        System.out.println("Child");
    }
}

public class Main {
    public static void main(String[] args) {
        Parent p = new Child();
        p.show();
    }
}
  • A) "Parent" が出力される
  • B) "Child" が出力される
  • C) コンパイルエラーが発生する
  • D) ランタイムエラーが発生する

正解は B) "Child" が出力される です。オーバーライドされたメソッドが実行時に実際のオブジェクトのタイプによって決定されます。

問題94: 可変長引数

        display("Hello", "World");
    }

  • A) "Hello" が出力される
  • B) "World" が出力される
  • C) "Hello World" が出力される
  • D) コンパイルエラーが発生する

正解は C) "Hello World" が出力される です。可変長引数により、複数の値を渡すことができます。

問題97: 例外処理

次のコードの実行結果はどうなりますか？

public class ExceptionExample {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("Arithmetic Exception");
        } finally {
            System.out.println("Finally block");
        }
    }
}
  • A) "Arithmetic Exception" が出力される
  • B) "Finally block" が出力される
  • C) "Arithmetic Exception" と "Finally block" が出力される
  • D) コンパイルエラーが発生する

正解は C) "Arithmetic Exception" と "Finally block" が出力される です。例外がキャッチされ、finallyブロックも実行されます。

問題98: シリアライズ

次のコードのコンパイル結果はどうなりますか？

import java.io.Serializable;

class MyClass implements Serializable {
    private static final long serialVersionUID = 1L;
    int id;
    String name;

    MyClass(int id, String name) {
        this.id = id;
        this.name = name;
    }
}

public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass(1, "Java");
        System.out.println(obj.id + " " + obj.name);
    }
}
  • A) "1 Java" が出力される
  • B) コンパイルエラーが発生する
  • C) ランタイムエラーが発生する
  • D) 何も出力されない

正解は A) "1 Java" が出力される です。クラスはSerializableを実装しており、通常の出力が行われます。

問題99: 文字列の比較

次のコードの実行結果はどうなりますか？

public class StringComparison {
    public static void main(String[] args) {
        String str1 = "Java";
        String str2 = new String("Java");
        System.out.println(str1 == str2);
        System.out.println(str1.equals(str2));
    }
}
  • A) true true が出力される
  • B) false true が出力される
  • C) true false が出力される
  • D) false false が出力される

正解は B) false true が出力される です。== 演算子は参照を比較し、equals メソッドは値を比較します。
