Java Silver 総合問題 - Part12

問題111: モジュールの基本

次のコードの正しいモジュール宣言はどれですか？

module mymodule {
    requires java.base;
    exports com.example.mypackage;
}
  • A) module mymodule { requires java.base; }
  • B) module mymodule { exports com.example.mypackage; }
  • C) module mymodule { requires java.base; exports com.example.mypackage; }
  • D) module mymodule { requires java.base; exports com.example.mypackage; requires java.sql; }
正解は C) module mymodule { requires java.base; exports com.example.mypackage; } です。モジュール宣言には、依存関係とエクスポートするパッケージが含まれます。

問題112: モジュール依存関係

次のモジュール宣言において、モジュールが依存している他のモジュールを指定するキーワードは何ですか？

module mymodule {
    ___ java.sql;
}
  • A) exports
  • B) requires
  • C) imports
  • D) includes
正解は B) requires です。モジュールが他のモジュールに依存する場合、requires キーワードを使用します。

問題113: モジュールエクスポート

次のコードにおいて、どのパッケージが他のモジュールに公開されますか？

module mymodule {
    exports com.example.mypublicpackage;
    requires java.base;
}
  • A) com.example.mypublicpackage
  • B) java.base
  • C) mymodule
  • D) すべてのパッケージ
正解は A) com.example.mypublicpackage です。exports キーワードを使って公開されたパッケージのみが他のモジュールからアクセス可能です。

問題114: モジュール名のルール

次のうち、正しいモジュール名のルールはどれですか？

  • A) モジュール名は数字で始めてはいけない
  • B) モジュール名にはアンダースコアを含めることができる
  • C) モジュール名にはスペースを含めることができる
  • D) モジュール名は大文字で始めなければならない
正解は A) モジュール名は数字で始めてはいけない です。モジュール名は識別子のルールに従い、数字で始めることはできません。

問題115: モジュールとパッケージ

次のモジュール宣言において、モジュール mymodule が提供するパッケージはどれですか？

module mymodule {
    exports com.example.mypackage;
    requires java.base;
}
  • A) mymodule
  • B) java.base
  • C) com.example.mypackage
  • D) すべてのパッケージ
正解は C) com.example.mypackage です。exports キーワードで公開されたパッケージが他のモジュールから利用可能になります。

問題116: モジュールの宣言方法

次のコードの正しいモジュール宣言はどれですか？

module mymodule {
    requires java.sql;
    exports com.example.package1;
    exports com.example.package2;
}
  • A) module mymodule { requires java.sql; }
  • B) module mymodule { exports com.example.package1; exports com.example.package2; }
  • C) module mymodule { requires java.sql; exports com.example.package1; exports com.example.package2; }
  • D) module mymodule { requires java.sql; exports com.example.package1; requires java.xml; }
正解は C) module mymodule { requires java.sql; exports com.example.package1; exports com.example.package2; } です。正しいモジュール宣言には依存関係とエクスポートパッケージの両方が含まれます。

問題117: モジュールのエクスポート制限

次のコードで、特定のモジュールにのみパッケージを公開するにはどうすればよいですか？

module mymodule {
    exports com.example.mypackage to anothermodule;
}
  • A) すべてのモジュールに公開される
  • B) anothermodule にのみ公開される
  • C) java.base にのみ公開される
  • D) どのモジュールにも公開されない
正解は B) anothermodule にのみ公開される です。exports ... to ... の形式で特定のモジュールにのみパッケージを公開できます。

問題118: 自動モジュール

次のうち、自動モジュールとして扱われるのはどれですか？

  • A) モジュール宣言がない JAR ファイル
  • B) module-info.java ファイルを含む JAR ファイル
  • C) 任意のクラスファイル
  • D) Maven プロジェクト
正解は A) モジュール宣言がない JAR ファイル です。モジュール宣言のない JAR ファイルは自動モジュールとして扱われ、依存関係を自動的に解決します。

問題119: サービスの使用

次のコードで、サービスのプロバイダをモジュールに指定するキーワードは何ですか？

module mymodule {
    uses com.example.myservice;
}
  • A) provides
  • B) uses
  • C) exports
  • D) requires
正解は B) uses です。モジュールが特定のサービスを使用することを示すためには uses キーワードを使用します。

問題120: サービスの提供

次のコードで、サービスを提供するクラスを指定するキーワードは何ですか？

module mymodule {
    provides com.example.myservice with com.example.myserviceimpl;
}
  • A) uses
  • B) requires
  • C) provides
  • D) exports
正解は C) provides です。モジュールが特定のサービスを提供するクラスを指定するためには provides キーワードを使用します。
