Java Silver 総合問題 - Part 6

問題51

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        int x = 10;
        x += 5;
        System.out.println(x);
    }
}
正解はB) 15です。
解説: x += 5; は x = x + 5; と同じ意味であり、xの値に5を加えた後に15が出力されます。

問題52

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        String str = "Java";
        str++;
        System.out.println(str);
    }
}
正解はC) コンパイルエラーです。
解説: String型の変数に対して++演算子を使用することはできません。コンパイルエラーが発生します。

問題53

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        int x = 5;
        x *= 2 + 3;
        System.out.println(x);
    }
}
正解はD) 25です。
解説: x *= 2 + 3; は x = x * (2 + 3); と同じ意味であり、xの値に10を乗じた後に25が出力されます。

問題54

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        int x = 10;
        System.out.println(x++);
        System.out.println(++x);
    }
}
正解はA) 10, 12です。
解説: x++ は後置インクリメントで、++x は前置インクリメントです。10が出力された後、xの値が11になり、それがインクリメントされて12が出力されます。

問題55

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        String str = "Hello";
        str += "World";
        System.out.println(str);
    }
}
正解はB) HelloWorldです。
解説: += 演算子はString型に対して連結を行います。"Hello" + "World" が連結されて"HelloWorld"が出力されます。

問題56

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        int x = 0;
        if (x++ < 1) {
            System.out.println("Java");
        } else {
            System.out.println("Python");
        }
    }
}
正解はA) Javaです。
解説: x++ は後置インクリメントで、if文の条件式はxの値が評価された後に1が加算されます。そのため、xの初期値が0なので条件は true となり、"Java"が出力されます。

問題57

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        boolean flag = true;
        if (flag) {
            System.out.println("Hello");
        } else {
            System.out.println("World");
        }
        System.out.println(flag);
    }
}
正解はB) Hello trueです。
解説: flag が true なので、if文の条件式は true となり、"Hello"が出力されます。その後、System.out.println(flag); によって true が出力されます。

問題59

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        int x = 10;
        int y = 20;
        System.out.println(x == y);
    }
}
正解はD) falseです。
解説: == 演算子はxとyの値を比較し、値が異なる場合は false を出力します。xは10、yは20なので、falseが出力されます。

問題60

次のJavaコードをコンパイルするとどうなりますか？


public class Test {
    public static void main(String[] args) {
        String str = null;
        if (str.equals("Java")) {
            System.out.println("Hello");
        } else {
            System.out.println("World");
        }
    }
}
正解はA) NullPointerExceptionが発生します。
解説: strがnullの場合、str.equals("Java") はNullPointerExceptionをスローします。そのため、このコードは実行時エラーとなります。
