問題101: java.lang.Math.powメソッド

次のコードの実行結果はどうなりますか？

public class Main {
    public static void main(String[] args) {
        double result = Math.pow(2, 3);
        System.out.println(result);
    }
}
  • A) 6.0 が出力される
  • B) 8.0 が出力される
  • C) 9.0 が出力される
  • D) コンパイルエラーが発生する
正解は B) 8.0 が出力される です。Math.pow(2, 3) は 2 の 3 乗を計算します。

問題103: Comparatorインタフェース

次のコードの実行結果はどうなりますか？

import java.util.*;

public class Main {
    public static void main(String[] args) {
        List list = Arrays.asList("banana", "apple", "cherry");
        Collections.sort(list, new Comparator() {
            @Override
            public int compare(String s1, String s2) {
                return s1.compareTo(s2);
            }
        });
        System.out.println(list);
    }
}
  • A) [apple, banana, cherry] が出力される
  • B) [cherry, banana, apple] が出力される
  • C) [banana, apple, cherry] が出力される
  • D) コンパイルエラーが発生する
正解は A) [apple, banana, cherry] が出力される です。Comparator インタフェースを使ってアルファベット順にソートしています。

問題105: Comparatorの比較方法変更

次のコードの実行結果はどうなりますか？

import java.util.*;

class Person {
    String name;
    int age;
    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

public class Main {
    public static void main(String[] args) {
        List list = Arrays.asList(new Person("Alice", 30), new Person("Bob", 25));
        Collections.sort(list, Comparator.comparingInt(p -> p.age));
        for (Person p : list) {
            System.out.println(p.name);
        }
    }
}
  • A) Alice Bob が出力される
  • B) Bob Alice が出力される
  • C) コンパイルエラーが発生する
  • D) ランタイムエラーが発生する
正解は B) Bob Alice が出力される です。Comparator.comparingInt を使用して年齢順にソートしています。
