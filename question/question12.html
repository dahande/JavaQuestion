Java Silver 総合問題 - Part 3

問題11

Javaで使用される基本的なデータ型は何ですか？

  • A) int, float, double, char, boolean
  • B) integer, float, double, character, boolean
  • C) int, float, double, char, string
  • D) integer, float, double, char, boolean
正解はA) int, float, double, char, booleanです。
解説: Javaの基本的なデータ型は、プリミティブ型として知られており、これらはメモリ上に直接の値を保持します。

問題12

サブクラスでスーパークラスのメソッドを上書きすることを何と呼びますか？

  • A) メソッドオーバーライド
  • B) メソッドオーバーロード
  • C) メソッドインターフェース
  • D) メソッド継承
正解はA) メソッドオーバーライドです。
解説: サブクラスでスーパークラスの同じシグネチャを持つメソッドを再定義することをメソッドオーバーライドと呼びます。

問題13

Javaのプリミティブ型に対応するラッパークラスは何ですか？

  • A) Integer, Float, Double, Char, Boolean
  • B) Int, Float, Double, Character, Boolean
  • C) Integer, Float, Double, Character, String
  • D) Int, Float, Double, Char, Boolean
正解はA) Integer, Float, Double, Character, Booleanです。
解説: ラッパークラスは、プリミティブ型のデータをオブジェクトとして扱うためのクラスです。

問題14

Javaで配列を宣言する際に使用する記号は何ですか？

  • A) ( )
  • B) [ ]
  • C) { }
  • D) < >
正解はB) [ ]です。
解説: 配列を宣言する際には、変数の型に続いて角括弧 [ ] を使用します。

問題15

Javaで指定された回数だけ繰り返し処理を行うために使用する制御構造は何ですか？

  • A) if文
  • B) whileループ
  • C) forループ
  • D) switch文
正解はC) forループです。
解説: forループは、指定された回数だけ処理を繰り返すために使用されます。

問題16

Javaのswitch文で、case節の最後には何が必要ですか？

  • A) 改行
  • B) ブレーク
  • C) コロン
  • D) ダブルクォーテーション
正解はB) ブレークです。
解説: switch文の各case節の最後には、break文または他の制御文（returnやthrow）を使用して制御を終了させる必要があります。

問題17

Javaで型安全なコレクションを実現するために使用される機能は何ですか？

  • A) 反復子
  • B) インタフェース
  • C) ジェネリクス
  • D) ラムダ式
正解はC) ジェネリクスです。
解説: ジェネリクスを使用すると、コレクションやメソッドなどで使用するオブジェクトの型を指定し、型安全性を確保できます。

問題18

Javaで関数型インタフェースをシンプルに実装するために使用される機能は何ですか？

  • A) ジェネリクス
  • B) 反復子
  • C) ラムダ式
  • D) インターフェース
正解はC) ラムダ式です。
解説: ラムダ式を使用すると、関数型インタフェースのインスタンスをシンプルに生成できます。

問題19

Javaでコレクションを処理するための新しいAPIは何ですか？

  • A) ジェネリクス
  • B) ラムダ式
  • C) 反復子
  • D) ストリームAPI
正解はD) ストリームAPIです。
解説: ストリームAPIを使用すると、コレクションを簡潔に操作して処理することができます。

問題20

Javaで複数のスレッドを同時に実行するための機能は何ですか？

  • A) ジェネリクス
  • B) ラムダ式
  • C) ストリームAPI
  • D) マルチスレッド処理
正解はD) マルチスレッド処理です。
解説: Javaでは、複数のスレッドを使用してプログラムを並行処理することができます。
