問題81: コンストラクタの継承

次のコードの実行結果はどうなりますか？

class Parent {
    Parent() {
        System.out.println("Parent Constructor");
    }
}

class Child extends Parent {
    Child() {
        System.out.println("Child Constructor");
    }
}

public class Main {
    public static void main(String[] args) {
        Child c = new Child();
    }
}
  • A) "Parent Constructor" が出力される
  • B) "Child Constructor" が出力される
  • C) "Parent Constructor" と "Child Constructor" が出力される
  • D) コンパイルエラーが発生する
正解は C) "Parent Constructor" と "Child Constructor" が出力される です。子クラスのコンストラクタが呼ばれる際に、親クラスのコンストラクタも自動的に呼び出されます。

問題82: メソッドのオーバーロード

次のコードの実行結果はどうなりますか？

class OverloadExample {
    void display(int a) {
        System.out.println("Integer: " + a);
    }

    void display(String a) {
        System.out.println("String: " + a);
    }
}

public class Main {
    public static void main(String[] args) {
        OverloadExample oe = new OverloadExample();
        oe.display(10);
        oe.display("Hello");
    }
}
  • A) "Integer: 10" が出力される
  • B) "String: Hello" が出力される
  • C) "Integer: 10" と "String: Hello" が出力される
  • D) コンパイルエラーが発生する
正解は C) "Integer: 10" と "String: Hello" が出力される です。メソッドのオーバーロードにより、異なるパラメータ型でメソッドが呼び出されています。

問題85: パッケージとimport文

import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {

        list.add("Hello");

        for (String s : list) {
            System.out.println(s);
        }
    }

  • A) "Hello" が出力される
  • B) "World" が出力される
  • C) "Hello" と "World" が出力される
  • D) コンパイルエラーが発生する

    }

        sc.display();

問題87: try-with-resourcesステートメント

import java.io.*;

public class Main {

        try (BufferedReader br = new BufferedReader(new FileReader("test.txt"))) {

        } catch (IOException e) {
            System.out.println("IOException caught");
        }
    }
}
  • A) ファイルの最初の行が出力される
  • B) "IOException caught" が出力される

  • D) 何も出力されない
正解は A) ファイルの最初の行が出力される です。try-with-resourcesステートメントにより、リソースが自動的にクローズされます。

    public static void main(String[] args) {

問題90: メソッド参照

次のコードの実行結果はどうなりますか？

import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        String[] strings = {"C", "A", "B"};
        Arrays.sort(strings, String::compareToIgnoreCase);
        for (String s : strings) {
            System.out.println(s);
        }
    }
}
  • A) "A", "B", "C" の順に出力される
  • B) "C", "A", "B" の順に出力される
  • C) コンパイルエラーが発生する
  • D) ランタイムエラーが発生する
正解は A) "A", "B", "C" の順に出力される です。String::compareToIgnoreCase を使用して配列をソートしています。
