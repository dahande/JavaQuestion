問題121: Consumerインターフェース

次のコードの実行結果はどうなりますか？

import java.util.function.Consumer;

public class Main {
    public static void main(String[] args) {
        Consumer consumer = s -> System.out.println(s.toUpperCase());
        consumer.accept("java");
    }
}
  • A) java が出力される
  • B) JAVA が出力される
  • C) コンパイルエラーが発生する
  • D) ランタイムエラーが発生する
正解は B) JAVA が出力される です。Consumer は渡された引数を大文字に変換して出力しています。

問題125: Consumerインターフェースの合成

次のコードの実行結果はどうなりますか？

import java.util.function.Consumer;

public class Main {
    public static void main(String[] args) {
        Consumer consumer1 = s -> System.out.print(s.toLowerCase());
        Consumer consumer2 = s -> System.out.print(s.toUpperCase());
        Consumer consumer = consumer1.andThen(consumer2);
        consumer.accept("Java");
    }
}
  • A) javaJAVA が出力される
  • B) JavaJAVA が出力される
  • C) JAVAjava が出力される
  • D) コンパイルエラーが発生する
正解は A) javaJAVA が出力される です。andThen メソッドで consumer1 の後に consumer2 が実行されます。

問題127: Predicateインターフェースの合成

次のコードの実行結果はどうなりますか？

import java.util.function.Predicate;

public class Main {
    public static void main(String[] args) {
        Predicate predicate1 = s -> s.length() > 3;
        Predicate predicate2 = s -> s.contains("a");
        Predicate predicate = predicate1.and(predicate2);
        System.out.println(predicate.test("Java"));
    }
}
  • A) true が出力される
  • B) false が出力される
  • C) コンパイルエラーが発生する
  • D) ランタイムエラーが発生する
正解は A) true が出力される です。Predicate は文字列の長さが3を超え、かつ 'a' を含む場合に true を返します。

問題129: Consumerインターフェースの例外処理

次のコードの実行結果はどうなりますか？

import java.util.function.Consumer;

public class Main {
    public static void main(String[] args) {
        Consumer consumer = s -> {
            if (s == null) throw new IllegalArgumentException("Argument is null");
            System.out.println(s.toUpperCase());
        };
        consumer.accept(null);
    }
}
  • A) null が出力される
  • B) NULL が出力される
  • C) IllegalArgumentException がスローされる
  • D) コンパイルエラーが発生する
正解は C) IllegalArgumentException がスローされる です。Consumer は引数が null の場合に例外をスローします。
