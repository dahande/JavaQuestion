Javaの主要な関数型インターフェースの解説

このページでは、Javaの主要な関数型インターフェースであるConsumer、Function、Supplier、およびPredicateについて説明します。

Consumer

Consumerは、単一の入力引数を受け取り、結果を返さない操作を行う関数型インターフェースです。典型的には入力引数に対して副作用を持つ操作を実行します。

import java.util.function.Consumer;

public class Main {
    public static void main(String[] args) {
        Consumer<String> consumer = (s) -> System.out.println(s);
        consumer.accept("Hello, Consumer!"); // 出力: Hello, Consumer!
    }
}

Function

Functionは、単一の入力引数を受け取り、それに基づいて結果を返す操作を行う関数型インターフェースです。

import java.util.function.Function;

public class Main {
    public static void main(String[] args) {
        Function<String, Integer> function = (s) -> s.length();
        int length = function.apply("Hello, Function!"); // 出力: 15
        System.out.println(length);
    }
}

Supplier

Supplierは、引数を受け取らずに結果を生成する操作を行う関数型インターフェースです。

import java.util.function.Supplier;

public class Main {
    public static void main(String[] args) {
        Supplier<String> supplier = () -> "Hello, Supplier!";
        String message = supplier.get();
        System.out.println(message); // 出力: Hello, Supplier!
    }
}

Predicate

Predicateは、単一の入力引数を受け取り、それに対して真偽値を返す操作を行う関数型インターフェースです。典型的には引数に対して条件をテストします。

import java.util.function.Predicate;

public class Main {
    public static void main(String[] args) {
        Predicate<String> predicate = (s) -> s.isEmpty();
        boolean result = predicate.test(""); // 出力: true
        System.out.println(result);
    }
}
