java.util.Comparator インタフェース

java.util.Comparator インタフェースは、オブジェクトの順序付けを定義するために使用されます。このインタフェースは、2つのオブジェクトを比較するためのメソッドを提供します。

compare メソッド

compare メソッドは、2つのオブジェクトを比較し、その順序を示す整数を返します。メソッドシグネチャは以下の通りです:

int compare(T o1, T o2)

返り値の意味は以下の通りです:

  • 負の整数: o1 が o2 よりも小さい
  • ゼロ: o1 が o2 と等しい
  • 正の整数: o1 が o2 よりも大きい

例: カスタムコンパレータの実装


import java.util.Comparator;

class Person {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

class AgeComparator implements Comparator<Person> {
    @Override
    public int compare(Person p1, Person p2) {
        return Integer.compare(p1.age, p2.age);
    }
}

public class ComparatorExample {
    public static void main(String[] args) {
        Person person1 = new Person("Alice", 30);
        Person person2 = new Person("Bob", 25);
        Comparator<Person> comparator = new AgeComparator();
        int result = comparator.compare(person1, person2);
        System.out.println("Comparison result: " + result);
    }
}

この例では、Person クラスのインスタンスを年齢で比較するために AgeComparator を実装しています。

compareTo メソッドとの違い

compareTo メソッドは Comparable インタフェースに属し、オブジェクト自身と他のオブジェクトを比較するために使用されます。一方、Comparator インタフェースの compare メソッドは、外部から2つのオブジェクトを比較するために使用されます。

Comparable インタフェースの例


class Person implements Comparable<Person> {
    String name;
    int age;

    Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @Override
    public int compareTo(Person other) {
        return Integer.compare(this.age, other.age);
    }
}

public class ComparableExample {
    public static void main(String[] args) {
        Person person1 = new Person("Alice", 30);
        Person person2 = new Person("Bob", 25);
        int result = person1.compareTo(person2);
        System.out.println("Comparison result: " + result);
    }
}

この例では、Person クラス自体が Comparable インタフェースを実装し、年齢での比較を提供しています。

まとめ

Comparator インタフェースは、オブジェクトの順序付けを外部から定義するための便利なツールです。これに対し、Comparable インタフェースはオブジェクト自身の順序付けを定義します。これらのインタフェースを使用することで、柔軟な比較ロジックを実装することができます。
