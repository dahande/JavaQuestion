Javaのオーバーライドの例

オーバーライドとは？

Javaにおけるオーバーライド（override）とは、スーパークラス（親クラス）で定義されたメソッドを、サブクラス（子クラス）で再定義することです。オーバーライドすることで、親クラスのメソッドの振る舞いを子クラスで変更することができます。

コード例

以下は、親クラスと子クラスを使ってオーバーライドを示したJavaのコード例です。

// 親クラスの定義
class Animal {
    void sound() {
        System.out.println("動物の鳴き声");
    }
}

// 子クラスの定義
class Dog extends Animal {
    // 親クラスのメソッドをオーバーライド
    @Override
    void sound() {
        System.out.println("ワンワン");
    }
}

// メインクラスの定義
public class Main {
    public static void main(String[] args) {
        Animal myDog = new Dog();
        myDog.sound(); // 出力: "ワンワン"
    }
}

解説

このコードでは、`Animal` クラスが基本クラス（親クラス）として定義されており、その中に `sound()` メソッドがあります。`Dog` クラスは `Animal` クラスを継承しており、`sound()` メソッドをオーバーライドしています。

メインクラスでは、`Animal` 型の変数 `myDog` に `Dog` クラスのインスタンスを代入し、`myDog.sound()` を呼び出すことで、オーバーライドされた `Dog` クラスの `sound()` メソッドが実行され、「ワンワン」という出力が表示されます。
