Javaのインタフェースの例

インタフェースとは？

Javaにおけるインタフェース（interface）とは、クラスが実装すべきメソッドを定義するための完全に抽象的な型です。インタフェースはメソッドの宣言だけを含み、具体的な実装は持ちません。

コード例

以下は、インタフェースとそれを実装するクラスを示したJavaのコード例です。

// インタフェースの定義
public interface A {
    // デフォルトメソッドの定義
    default void sample() {
        System.out.println("sample");
    }
}

// インタフェースを実装するクラスの定義
public class B implements A {
    // インタフェースのメソッドをオーバーライドする必要はありません
}

// メインクラスの定義
public class Main {
    public static void main(String[] args) {
        // インタフェース型の変数にクラスのインスタンスを代入
        A a = new B();
        // インタフェースのメソッドを呼び出す
        a.sample(); // 出力: "sample"
    }
}

解説

このコードでは、`A` インタフェースが `sample()` というデフォルトメソッドを定義しています。クラス `B` は `A` インタフェースを実装しており、`sample()` メソッドをオーバーライドしていませんが、`A` インタフェースのデフォルト実装を使用できます。

メインクラスでは、インタフェース型の変数 `a` に `B` クラスのインスタンスを代入し、`a.sample()` を呼び出すことで、`sample` メソッドのデフォルト実装が呼び出され、「sample」という出力が表示されます。

インタフェースの多重実現

Javaで２つ以上のクラスを同時に継承する多重継承が禁止されていますが、インタフェースの多重実現は可能である。しかし多重実現で注意しなければいけないのが、同じシグニチャのデフォルトメゾットの存在はエラーが出ます
