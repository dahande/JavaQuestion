Javaのアクセス修飾子

public

public修飾子は、クラス、メソッド、フィールドがどこからでもアクセスできることを示します。プロジェクトの他のパッケージやクラスからもアクセス可能です。

// public クラス
public class MyClass {
    // public フィールド
    public int myNumber;

    // public メソッド
    public void myMethod() {
        System.out.println("This is a public method.");
    }
}

// 他のクラスからのアクセス
public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        obj.myNumber = 10;
        obj.myMethod(); // 出力: This is a public method.
    }
}

private

private修飾子は、クラス内でのみアクセスできることを示します。他のクラスやパッケージからはアクセスできません。

// private フィールドとメソッド
public class MyClass {
    private int myNumber;

    private void myMethod() {
        System.out.println("This is a private method.");
    }

    public void accessPrivateMethod() {
        myMethod(); // クラス内ではアクセス可能
    }
}

// 他のクラスからのアクセス
public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        // obj.myNumber = 10; // エラー: myNumberはprivate
        // obj.myMethod(); // エラー: myMethodはprivate
        obj.accessPrivateMethod(); // 出力: This is a private method.
    }
}

protected

protected修飾子は、同じパッケージ内のクラスおよびサブクラスからアクセスできることを示します。

// protected フィールドとメソッド
public class MyClass {
    protected int myNumber;

    protected void myMethod() {
        System.out.println("This is a protected method.");
    }
}

// サブクラスからのアクセス
public class SubClass extends MyClass {
    public void accessProtected() {
        myNumber = 20;
        myMethod(); // 出力: This is a protected method.
    }
}

// 他のクラスからのアクセス
public class Main {
    public static void main(String[] args) {
        SubClass obj = new SubClass();
        obj.accessProtected();
    }
}

デフォルト（パッケージプライベート）

アクセス修飾子を指定しない場合、そのフィールドやメソッドはデフォルトで同じパッケージ内からのみアクセス可能になります。

// デフォルトアクセスのフィールドとメソッド
public class MyClass {
    int myNumber; // デフォルト（パッケージプライベート）
    
    void myMethod() { // デフォルト（パッケージプライベート）
        System.out.println("This is a package-private method.");
    }
}

// 同じパッケージ内のクラスからのアクセス
public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        obj.myNumber = 30;
        obj.myMethod(); // 出力: This is a package-private method.
    }
}
