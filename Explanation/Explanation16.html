Java `java.time` クラス

概要

`java.time`パッケージは、Java 8以降で導入された、日時の操作に特化したクラス群です。従来の`java.util.Date`や`java.util.Calendar`に代わり、より直感的で使いやすいAPIを提供します。

主要クラス

  • LocalDate
  • LocalTime
  • LocalDateTime
  • ZonedDateTime

LocalDate

`LocalDate`は、年月日を表すクラスです。時間やタイムゾーンの情報は含まれません。

import java.time.LocalDate;

public class Main {
    public static void main(String[] args) {
        LocalDate date = LocalDate.now();
        System.out.println("現在の日付: " + date);

        LocalDate specificDate = LocalDate.of(2024, 7, 1);
        System.out.println("特定の日付: " + specificDate);
    }
}

LocalTime

`LocalTime`は、時間（時、分、秒、ナノ秒）を表すクラスです。日付やタイムゾーンの情報は含まれません。

import java.time.LocalTime;

public class Main {
    public static void main(String[] args) {
        LocalTime time = LocalTime.now();
        System.out.println("現在の時刻: " + time);

        LocalTime specificTime = LocalTime.of(14, 30, 15);
        System.out.println("特定の時刻: " + specificTime);
    }
}

LocalDateTime

`LocalDateTime`は、年月日と時間を合わせて表すクラスです。タイムゾーンの情報は含まれません。

import java.time.LocalDateTime;

public class Main {
    public static void main(String[] args) {
        LocalDateTime dateTime = LocalDateTime.now();
        System.out.println("現在の日時: " + dateTime);

        LocalDateTime specificDateTime = LocalDateTime.of(2024, 7, 1, 14, 30, 15);
        System.out.println("特定の日時: " + specificDateTime);
    }
}

ZonedDateTime

`ZonedDateTime`は、日時とタイムゾーンを合わせて表すクラスです。

import java.time.ZonedDateTime;
import java.time.ZoneId;

public class Main {
    public static void main(String[] args) {
        ZonedDateTime zonedDateTime = ZonedDateTime.now();
        System.out.println("現在の日時（タイムゾーン付き）: " + zonedDateTime);

        ZonedDateTime specificZonedDateTime = ZonedDateTime.of(2024, 7, 1, 14, 30, 15, 0, ZoneId.of("Asia/Tokyo"));
        System.out.println("特定の日時（タイムゾーン付き）: " + specificZonedDateTime);
    }
}
