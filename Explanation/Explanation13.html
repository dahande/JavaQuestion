java.lang.Math クラス

java.lang.Math クラスは、数学的な演算を行うためのメソッドを提供するクラスです。このクラスには、基本的な算術演算や三角関数、対数関数などが含まれています。すべてのメソッドは静的メソッドであり、インスタンスを作成せずに使用することができます。

主要メソッドの紹介

pow メソッド

pow メソッドは、指定された数のべき乗を計算します。具体的には、Math.pow(double a, double b) は a の b 乗を返します。

double result = Math.pow(2, 3); // 2 の 3 乗、すなわち 8.0 が返されます。

例:


public class MathExample {
    public static void main(String[] args) {
        double base = 2.0;
        double exponent = 3.0;
        double result = Math.pow(base, exponent);
        System.out.println(base + " の " + exponent + " 乗は " + result + " です。");
    }
}

sqrt メソッド

sqrt メソッドは、指定された数の平方根を計算します。具体的には、Math.sqrt(double a) は a の平方根を返します。

double result = Math.sqrt(16); // 16 の平方根、すなわち 4.0 が返されます。

例:


public class MathExample {
    public static void main(String[] args) {
        double value = 16.0;
        double result = Math.sqrt(value);
        System.out.println(value + " の平方根は " + result + " です。");
    }
}

その他のメソッド

java.lang.Math クラスには、その他にも多くの便利なメソッドがあります。例えば、以下のようなものがあります:

  • abs(double a): 絶対値を返します。
  • max(double a, double b): 2つの値のうち大きい方を返します。
  • min(double a, double b): 2つの値のうち小さい方を返します。
  • random(): 0.0 以上 1.0 未満のランダムな値を返します。
  • round(double a): 四捨五入した値を返します。

まとめ

java.lang.Math クラスは、数学的な計算を効率的に行うための豊富なメソッドを提供しています。これらのメソッドを活用することで、複雑な計算を簡単に実装することができます。
