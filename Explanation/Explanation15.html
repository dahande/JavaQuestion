Java Characterクラスの解説

Characterクラスとラッパークラス

Characterクラスは、プリミティブ型のcharデータをオブジェクトで扱うためのラッパークラスです。ラッパークラスは、プリミティブデータ型をオブジェクトとして扱うために使用されるクラスであり、Characterクラスは文字に関連する多くの便利なメソッドを提供します。

isAlphabeticメソッド

isAlphabeticメソッドは、指定された文字がアルファベットかどうかを判定します。このメソッドは、文字がアルファベットであればtrueを返し、そうでなければfalseを返します。 


public static boolean isAlphabetic(int codePoint)

例:


public class CharacterExample {
    public static void main(String[] args) {
        char ch = 'A';
        boolean result = Character.isAlphabetic(ch);
        System.out.println(ch + " is alphabetic: " + result); // 出力: A is alphabetic: true
    }
}

isDigitメソッド

isDigitメソッドは、指定された文字が数字かどうかを判定します。このメソッドは、文字が数字であればtrueを返し、そうでなければfalseを返します。 


public static boolean isDigit(char ch)

例:


public class CharacterExample {
    public static void main(String[] args) {
        char ch = '5';
        boolean result = Character.isDigit(ch);
        System.out.println(ch + " is digit: " + result); // 出力: 5 is digit: true
    }
}

isLowerCaseメソッド

isLowerCaseメソッドは、指定された文字が小文字かどうかを判定します。このメソッドは、文字が小文字であればtrueを返し、そうでなければfalseを返します。 


public static boolean isLowerCase(char ch)

例:


public class CharacterExample {
    public static void main(String[] args) {
        char ch = 'a';
        boolean result = Character.isLowerCase(ch);
        System.out.println(ch + " is lowercase: " + result); // 出力: a is lowercase: true
    }
}

まとめ

Characterクラスは、文字に関連する多くの便利なメソッドを提供します。これらのメソッドを使用することで、文字の判定や変換を簡単に行うことができます。isAlphabetic、isDigit、isLowerCaseメソッドは、その一例です。
