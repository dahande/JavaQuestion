Java 例外処理の解説

概要

例外処理は、プログラムの実行中に発生するエラーを管理し、適切に対処するためのメカニズムです。Javaでは、try、catch、finallyブロックを使用して例外を処理します。

主要なクラスとインターフェース

Javaの例外処理に関する主要なクラスとインターフェースは以下の通りです:

  • Throwable: すべてのエラーや例外のスーパークラスです。
  • Exception: 一般的な例外を表すクラスです。
  • RuntimeException: 実行時例外を表すクラスです。
  • Error: アプリケーションが処理できない重大なエラーを表すクラスです。

例外処理の方法

Javaでは、以下のように例外を処理します:


try {
    // 例外が発生する可能性のあるコード
} catch (ExceptionType e) {
    // 例外をキャッチして処理するコード
} finally {
    // 例外の発生有無にかかわらず実行されるコード
}

サンプルコード


public class Main {
    public static void main(String[] args) {
        try {
            int[] numbers = {1, 2, 3};
            System.out.println(numbers[5]); // 例外が発生するコード
        } catch (ArrayIndexOutOfBoundsException e) {
            System.out.println("配列のインデックスが範囲外です: " + e.getMessage());
        } finally {
            System.out.println("例外処理が完了しました。");
        }

        try {
            int result = divide(10, 0); // 例外が発生する可能性のあるメソッドの呼び出し
        } catch (ArithmeticException e) {
            System.out.println("算術演算エラー: " + e.getMessage());
        }
    }

    public static int divide(int a, int b) {
        return a / b; // bが0の場合に例外が発生
    }
}

このサンプルコードでは、配列の範囲外アクセスと算術演算エラーを例外処理する方法を示しています。
