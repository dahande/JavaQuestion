Javaのオーバーロードの例

オーバーロードとは？

Javaにおけるオーバーロード（overload）とは、同じクラス内で同じ名前のメソッドを複数定義し、それぞれ異なる引数リスト（引数の型や数）を持つことを指します。オーバーロードにより、同じ名前のメソッドで異なる引数を受け取ることができます。

コード例

以下は、オーバーロードを示したJavaのコード例です。

// クラスの定義
public class Calculator {
    // メソッドのオーバーロード
    public int add(int a, int b) {
        return a + b;
    }

    public double add(double a, double b) {
        return a + b;
    }

    public int add(int a, int b, int c) {
        return a + b + c;
    }
}

// メインクラスの定義
public class Main {
    public static void main(String[] args) {
        Calculator calc = new Calculator();

        // オーバーロードされたメソッドの呼び出し
        System.out.println("2 + 3 = " + calc.add(2, 3)); // 出力: 2 + 3 = 5
        System.out.println("2.5 + 3.5 = " + calc.add(2.5, 3.5)); // 出力: 2.5 + 3.5 = 6.0
        System.out.println("1 + 2 + 3 = " + calc.add(1, 2, 3)); // 出力: 1 + 2 + 3 = 6
    }
}

解説

このコードでは、`Calculator` クラスが `add` メソッドを3つ定義しています。それぞれの `add` メソッドは、異なる引数リストを持ちます。

  • 最初の `add` メソッドは2つの `int` 型引数を受け取り、それらの和を返します。
  • 次の `add` メソッドは2つの `double` 型引数を受け取り、それらの和を返します。
  • 最後の `add` メソッドは3つの `int` 型引数を受け取り、それらの和を返します。

メインクラスの `main` メソッドでは、これらのオーバーロードされた `add` メソッドを呼び出し、それぞれの引数リストに基づいて適切なメソッドが実行される様子が示されています。
