Javaのメソッドとコンストラクタの違い

メソッド（Method）

メソッドはクラス内で定義される処理のブロックであり、特定の動作や処理を実行するために使用されます。メソッドは以下の特徴を持ちます：

  • メソッドは何度でも呼び出すことができ、同じ処理を再利用することができます。
  • メソッドはパラメータを受け取り、必要に応じて戻り値を返すことができます。
  • 例えば、次のコードは「Hello, World!」を出力するメソッドの例です：
public class MyClass {
    // メソッドの定義
    public void sayHello() {
        System.out.println("Hello, World!");
    }

    // メインメソッド（プログラムのエントリーポイント）
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        obj.sayHello(); // 出力: Hello, World!
    }
}

コンストラクタ（Constructor）

コンストラクタはクラスのインスタンスが作成される際に自動的に呼び出される特殊なメソッドです。主な目的は、インスタンス変数の初期化や必要な設定を行うことです。コンストラクタは以下の特徴を持ちます：

  • クラスと同じ名前であり、戻り値を持ちません。
  • コンストラクタはnewキーワードを使用してインスタンスが作成されるときに呼び出されます。
  • 次の例は、コンストラクタの定義例です：
public class Person {
    private String name;
    private int age;

    // コンストラクタの定義
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // インスタンスの情報を表示するメソッド
    public void displayInfo() {
        System.out.println("Name: " + name + ", Age: " + age);
    }

    // メインメソッド（プログラムのエントリーポイント）
    public static void main(String[] args) {
        // コンストラクタを使用してインスタンスを作成
        Person person1 = new Person("Alice", 30);
        person1.displayInfo(); // 出力: Name: Alice, Age: 30

        Person person2 = new Person("Bob", 25);
        person2.displayInfo(); // 出力: Name: Bob, Age: 25
    }
}
