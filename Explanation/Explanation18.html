Java Arrays クラスの解説

概要

Arraysクラスは、配列の操作を支援する静的メソッドを提供するユーティリティクラスです。配列のソート、検索、比較、および操作を簡単に行うことができます。

mismatch メソッド

mismatchメソッドは、2つの配列間の最初の不一致を検出します。このメソッドは、2つの配列の最初の不一致要素のインデックスを返します。不一致が見つからなかった場合は、-1を返します。

シグネチャ:

  • static int mismatch(int[] a, int[] b)
  • static int mismatch(long[] a, long[] b)
  • static int mismatch(double[] a, double[] b)
  • その他のプリミティブ型およびオブジェクト型のオーバーロード

compare メソッド

compareメソッドは、2つの配列を辞書順で比較します。このメソッドは、最初の配列が2番目の配列よりも小さい場合は負の値、等しい場合は0、大きい場合は正の値を返します。

シグネチャ:

  • static int compare(int[] a, int[] b)
  • static int compare(long[] a, long[] b)
  • static int compare(double[] a, double[] b)
  • その他のプリミティブ型およびオブジェクト型のオーバーロード

サンプルコード


import java.util.Arrays;

public class Main {
    public static void main(String[] args) {
        // 配列の作成
        int[] array1 = {1, 2, 3, 4, 5};
        int[] array2 = {1, 2, 0, 4, 5};
        int[] array3 = {1, 2, 3, 4, 5};
        
        // mismatch メソッドの使用例
        int mismatchIndex = Arrays.mismatch(array1, array2);
        System.out.println("Mismatch index between array1 and array2: " + mismatchIndex);
        
        // compare メソッドの使用例
        int comparisonResult = Arrays.compare(array1, array3);
        if (comparisonResult < 0) {
            System.out.println("array1 is less than array3");
        } else if (comparisonResult == 0) {
            System.out.println("array1 is equal to array3");
        } else {
            System.out.println("array1 is greater than array3");
        }
    }
}

このサンプルコードでは、mismatchメソッドを使用して2つの配列間の最初の不一致を検出し、compareメソッドを使用して2つの配列を辞書順で比較しています。
