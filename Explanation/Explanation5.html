Java Silver 基礎知識

Javaの基本構文

Javaの基本構文について学びます。変数、データ型、演算子、制御構文などの基本的な要素を理解することが重要です。

// 変数の宣言と初期化
int number = 10;
String message = "Hello, World!";

// 制御構文の例
if (number > 5) {
    System.out.println(message);
} else {
    System.out.println("Number is less than or equal to 5");
}

オブジェクト指向の概念

Javaはオブジェクト指向プログラミング言語です。クラス、オブジェクト、継承、ポリモーフィズムなどの基本的な概念を理解しましょう。

// クラスとオブジェクトの例
public class Dog {
    String name;
    int age;

    public void bark() {
        System.out.println(name + " says Woof!");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.name = "Buddy";
        myDog.age = 3;
        myDog.bark(); // 出力: Buddy says Woof!
    }
}

メソッド

メソッドは特定のタスクを実行するための関数です。Javaでは、メソッドはクラス内で定義され、クラスのオブジェクトによって呼び出されます。

// メソッドの定義と呼び出し
public class Calculator {
    public int add(int a, int b) {
        return a + b;
    }
}

public class Main {
    public static void main(String[] args) {
        Calculator calc = new Calculator();
        int result = calc.add(5, 3);
        System.out.println("結果: " + result); // 出力: 結果: 8
    }
}

配列

配列は同じデータ型の複数の要素を格納するためのデータ構造です。配列の宣言と初期化の方法について学びましょう。

// 配列の宣言と初期化
int[] numbers = {1, 2, 3, 4, 5};

// 配列の要素にアクセス
for (int i = 0; i < numbers.length; i++) {
    System.out.println("Number at index " + i + ": " + numbers[i]);
}

継承

継承は既存のクラスを基に新しいクラスを作成するための機能です。親クラスのメソッドやフィールドを子クラスに継承することができます。

// 継承の例
class Animal {
    public void eat() {
        System.out.println("食べる");
    }
}

class Dog extends Animal {
    public void bark() {
        System.out.println("吠える");
    }
}

public class Main {
    public static void main(String[] args) {
        Dog myDog = new Dog();
        myDog.eat(); // 出力: 食べる
        myDog.bark(); // 出力: 吠える
    }
}

インタフェース

インタフェースは、クラスが実装するメソッドの宣言のみを含む抽象型です。インタフェースを実装することで、クラスは特定のメソッドを必ず持つことが保証されます。

// インタフェースの定義
public interface Animal {
    void makeSound();
}

// インタフェースの実装
public class Dog implements Animal {
    public void makeSound() {
        System.out.println("ワンワン");
    }
}

public class Main {
    public static void main(String[] args) {
        Animal myDog = new Dog();
        myDog.makeSound(); // 出力: ワンワン
    }
}

例外処理

例外処理は、プログラムの実行中に発生するエラーを処理するための仕組みです。try-catchブロックを使用して例外をキャッチし、適切に処理します。

// 例外処理の例
public class Main {
    public static void main(String[] args) {
        try {
            int result = 10 / 0;
        } catch (ArithmeticException e) {
            System.out.println("エラー: 0で割ることはできません。");
        }
    }
}
