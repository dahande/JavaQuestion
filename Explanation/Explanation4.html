Javaのコンストラクタの例

コンストラクタとは？

Javaにおけるコンストラクタとは、オブジェクトのインスタンス生成時に呼び出される特殊なメソッドです。コンストラクタはクラス名と同じ名前を持ち、戻り値を持ちません。コンストラクタは通常、オブジェクトの初期化に使用されます。

コード例

以下は、コンストラクタを示したJavaのコード例です。

// クラスの定義
public class Person {
    String name;
    int age;

    // デフォルトコンストラクタ
    public Person() {
        this.name = "名無し";
        this.age = 0;
    }

    // 引数付きコンストラクタ
    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // メソッド
    public void displayInfo() {
        System.out.println("名前: " + name + ", 年齢: " + age);
    }
}

// メインクラスの定義
public class Main {
    public static void main(String[] args) {
        // デフォルトコンストラクタを使用してインスタンスを生成
        Person person1 = new Person();
        person1.displayInfo(); // 出力: 名前: 名無し, 年齢: 0

        // 引数付きコンストラクタを使用してインスタンスを生成
        Person person2 = new Person("田中太郎", 25);
        person2.displayInfo(); // 出力: 名前: 田中太郎, 年齢: 25
    }
}

解説

このコードでは、`Person` クラスが2つのコンストラクタを持っています。

  • デフォルトコンストラクタは引数を取らず、名前を "名無し" に、年齢を 0 に初期化します。
  • 引数付きコンストラクタは、名前と年齢を引数として受け取り、それに基づいてフィールドを初期化します。

メインクラスの `main` メソッドでは、これらのコンストラクタを使用して `Person` クラスのインスタンスを生成し、それぞれの初期化状態を `displayInfo` メソッドで出力しています。
