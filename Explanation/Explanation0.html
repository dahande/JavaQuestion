Javaのインスタンスの例

インスタンスとは？

Javaにおけるインスタンスとは、クラスを基にして生成された具体的なオブジェクトのことです。クラスは設計図として機能し、その設計図に基づいて生成された具体的なオブジェクトがインスタンスです。

コード例

以下は、インスタンスを生成し使用するJavaのコード例です。

// クラスの定義
public class Car {
    String brand;
    int speed;

    // コンストラクタ
    public Car(String brand, int speed) {
        this.brand = brand;
        this.speed = speed;
    }

    // メソッド
    public void displayInfo() {
        System.out.println("ブランド: " + brand + ", 速度: " + speed + " km/h");
    }

    // スピードアップメソッド
    public void speedUp(int increment) {
        speed += increment;
        System.out.println("新しい速度: " + speed + " km/h");
    }
}

// メインクラスの定義
public class Main {
    public static void main(String[] args) {
        // インスタンスの生成
        Car car1 = new Car("トヨタ", 120);
        car1.displayInfo(); // 出力: ブランド: トヨタ, 速度: 120 km/h

        // インスタンスのメソッドを呼び出す
        car1.speedUp(30); // 出力: 新しい速度: 150 km/h

        // 別のインスタンスを生成
        Car car2 = new Car("ホンダ", 100);
        car2.displayInfo(); // 出力: ブランド: ホンダ, 速度: 100 km/h
    }
}

解説

このコードでは、`Car` クラスが定義されています。このクラスは車のブランドと速度を表すフィールドを持ち、これらを初期化するためのコンストラクタと、情報を表示するための `displayInfo` メソッド、および速度を上げるための `speedUp` メソッドを持っています。

  • `Car` クラスのコンストラクタは、ブランドと速度を引数として受け取り、それらを対応するフィールドに設定します。
  • `displayInfo` メソッドは、車のブランドと速度をコンソールに出力します。
  • `speedUp` メソッドは、指定された値だけ速度を増加させ、新しい速度をコンソールに出力します。

`Main` クラスでは、`Car` クラスのインスタンスを2つ生成し、それぞれのインスタンスのメソッドを呼び出して情報を表示し、速度を変更しています。このように、クラスを基にした具体的なオブジェクト（インスタンス）を生成し、操作することができます。
