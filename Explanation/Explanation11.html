Javaのメソッドとその使用例

このページでは、Javaのメソッドの定義とその使用例について説明します。

サンプルコード

以下は、`Sample`クラスのメソッドを使ったサンプルコードです。

public class Main {
    public static void main(String[] args) {        
        Sample s = new Sample();
        String val = s.setValue("hello");
        s.getValue();
        System.out.println(val);
    }
}

public class Sample{
    private String num;
    private String value;
    
    // 値を設定するメソッド
    public void setValue(String value){
        this.value = value;
    }
    
    // 値を取得するメソッド
    public String getValue(){
        return this.value;
    }
}

解説

このサンプルコードでは、`Sample`クラスに2つのメソッドがあります:

  • setValue(String value): 引数として渡された文字列を`value`フィールドに設定します。
  • getValue(): `value`フィールドの値を返します。

修正後のコード

元のコードにはいくつかの問題がありました。例えば、setValue()メソッドは戻り値がないのに対し、呼び出し元で戻り値を受け取ろうとしていました。以下は修正後のコードです。

public class Main {
    public static void main(String[] args) {        
        Sample s = new Sample();
        s.setValue("hello");
        String val = s.getValue();
        System.out.println(val); // 出力: hello
    }
}

public class Sample{
    private String num;
    private String value;
    
    // 値を設定するメソッド
    public void setValue(String value){
        this.value = value;
    }
    
    // 値を取得するメソッド
    public String getValue(){
        return this.value;
    }
}

修正点の説明

修正後のコードでは、以下の点が改善されています:

  • setValue()メソッドはvoid型で戻り値がないため、その呼び出しはString val = s.setValue("hello");からs.setValue("hello");に変更されました。
  • getValue()メソッドを呼び出してその戻り値を変数valに格納し、System.out.println(val);で出力しています。
