Javaのラムダ式について

ラムダ式は、Java 8で導入された機能で、匿名関数（名前のない関数）を簡潔に表現するための構文です。ラムダ式は主にコレクションの操作やイベントハンドリングで使用され、コードをより簡潔で読みやすくします。

ラムダ式の基本構文

ラムダ式の基本構文は以下の通りです。

(引数) -> { 処理 }

例えば、引数を1つ取り、その引数を標準出力に表示するラムダ式は次のようになります。

(x) -> { System.out.println(x); }

簡単な例

リストの各要素を2倍にするラムダ式を使用した例を見てみましょう。

import java.util.Arrays;
import java.util.List;

public class LambdaExample {
    public static void main(String[] args) {
        List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);

        // ラムダ式を使用して各要素を2倍にする
        numbers.forEach(n -> System.out.println(n * 2));
    }
}

ラムダ式の利点

  • コードが簡潔になる
  • 匿名クラスの代替として使用できる
  • 関数型プログラミングのサポート

コード例: イベントハンドリング

ラムダ式はイベントハンドリングでもよく使用されます。例えば、ボタンのクリックイベントを処理する場合です。

import javax.swing.JButton;
import javax.swing.JFrame;

public class LambdaSwingExample {
    public static void main(String[] args) {
        JFrame frame = new JFrame("Lambda Example");
        JButton button = new JButton("Click Me");

        // ラムダ式を使用してクリックイベントを処理
        button.addActionListener(e -> System.out.println("Button Clicked"));

        frame.add(button);
        frame.setSize(300, 200);
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setVisible(true);
    }
}

Javaのラムダ式の例と解説

このページでは、ラムダ式を使用してJavaプログラムを簡潔に記述する方法を解説します。以下のコード例では、ラムダ式を用いて名前を表示するシンプルなアルゴリズムを実装しています。

コード例

public class Main {
public static void main(String[] args) {        
Algorithm algorithm = (name) -> {
    System.out.println("hello," + name);
};
Service s = new Service();
s.setLogic(algorithm);
s.doProcess("Lambda");
}
}

interface Algorithm {
void perform(String name);
}

class Service {
private Algorithm logic;

public void setLogic(Algorithm logic){
this.logic = logic;
}

public void doProcess(String name){
System.out.println("start");
this.logic.perform(name);
System.out.println("end");
}
}

コードの解説

このコードは、ラムダ式を使用して名前を表示するアルゴリズムを実装し、それをサービスクラスに設定して処理を実行する例です。

Mainクラス

Mainクラスはプログラムのエントリーポイントです。以下の部分でラムダ式を使ってAlgorithmインターフェースのメソッドを実装しています。

Algorithm algorithm = (name) -> {
System.out.println("hello," + name);
};

このラムダ式は、Algorithmインターフェースのperformメソッドを実装しています。

Algorithmインターフェース

Algorithmインターフェースは、名前を引数として受け取り、それを表示するメソッドperformを定義しています。

interface Algorithm {
void perform(String name);
}

Serviceクラス

Serviceクラスは、Algorithmを用いて処理を実行するクラスです。setLogicメソッドでラムダ式を設定し、doProcessメソッドで処理を実行します。

class Service {
private Algorithm logic;

public void setLogic(Algorithm logic){
this.logic = logic;
}

public void doProcess(String name){
System.out.println("start");
this.logic.perform(name);
System.out.println("end");
}
}

この例では、以下のように出力されます。

start
hello,Lambda
end
