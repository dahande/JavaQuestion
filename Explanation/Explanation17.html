Java ArrayList クラスの解説

概要

ArrayListクラスは、Javaコレクションフレームワークの一部であり、動的な配列のように機能します。サイズが可変であり、要素の追加、削除、検索などの操作が効率的に行えます。

コンストラクタ

  • ArrayList() - 初期容量が10の空のリストを作成します。
  • ArrayList(int initialCapacity) - 指定された初期容量を持つ空のリストを作成します。
  • ArrayList(Collection c) - 指定されたコレクションの要素を含むリストを作成します。

主なメソッド

  • boolean add(E e) - リストの末尾に指定された要素を追加します。
  • void add(int index, E element) - 指定された位置に要素を挿入します。
  • E get(int index) - 指定された位置にある要素を返します。
  • E remove(int index) - 指定された位置にある要素を削除し、その要素を返します。
  • boolean remove(Object o) - 指定された要素の最初の出現を削除します。
  • int size() - リストの要素数を返します。
  • boolean isEmpty() - リストが空である場合にtrueを返します。
  • void clear() - リストの全ての要素を削除します。
  • boolean contains(Object o) - リストに指定された要素が含まれている場合にtrueを返します。
  • int indexOf(Object o) - リスト内の指定された要素の最初の出現のインデックスを返します。

サンプルコード


import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        // ArrayListの作成
        ArrayList list = new ArrayList<>();

        // 要素の追加
        list.add("Apple");
        list.add("Banana");
        list.add("Cherry");

        // 指定した位置に要素を追加
        list.add(1, "Blueberry");

        // 要素の取得
        System.out.println("Index 2: " + list.get(2));

        // 要素の削除
        list.remove("Banana");
        System.out.println("After removing Banana: " + list);

        // サイズの取得
        System.out.println("Size: " + list.size());

        // 要素の確認
        System.out.println("Contains Cherry: " + list.contains("Cherry"));

        // リストのクリア
        list.clear();
        System.out.println("List after clear: " + list);
    }
}

このサンプルコードでは、ArrayListの基本的な操作（追加、挿入、取得、削除、確認、クリア）が示されています。
